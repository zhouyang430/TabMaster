# TabMaster
a Chrome extension that auto-groups, searches, and deduplicates your browser tabs so you never lose (or re-open) a doc again.

安装步骤：

下载项目到本地

1. 打开 Chrome 浏览器 ，在地址栏输入 chrome://extensions/ 并回车
2. 开启开发者模式 ：
   
   - 点击页面右上角的 "开发者模式" 开关，将其打开
3. 加载扩展程序 ：
   
   - 点击页面左上角的 "加载已解压的扩展程序" 按钮
   - 在文件选择器中，选择 /Users/bytedance/Documents/trae_projects/tabmaster-pro/dist 目录
   - 点击 "选择文件夹"
4. 开始使用 ：
   
   - 安装成功后，浏览器右上角会出现 TabMaster Pro 的图标
   - 在任意网页右下角会看到悬浮球，显示当前标签数量
   - 悬停悬浮球 300ms 即可展开预览面板
测试核心功能：

1. 悬浮球交互 ：悬停展开面板，点击收起，拖拽调整位置
2. 标签搜索 ：在面板顶部搜索框输入关键词实时过滤
3. 分类筛选 ：分类导航快速筛选特定类别标签
4. 一键去重 ：点击面板底部"一键去重"按钮清除重复标签
5. 设置中心 ：右键悬浮球或点击扩展图标打开设置页面
快捷键测试 ：

- Alt + Shift + T ：唤起/关闭预览面板
- Alt + Shift + D ：立即执行去重扫描
